How to use

- Visit chrome://net-export
- In "OPTIONS" set "Include raw bytes"
- Click "Start Logging to Disk"
- Visit chrome://policy
- Click "Reload policies"
- Go back to chrome://net-export and click "Stop logging"
- Upload file here :)
